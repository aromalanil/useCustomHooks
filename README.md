## 🧰 Installation

### Using npm or yarn 📦

```bash
# Using npm
npm i use-custom-hooks

# Using yarn
yarn add use-custom-hooks
```
## 📘 Available Hooks

- useStack
- useToggle
- useDebounce
- useDarkMode
- useGeoLocation
- useLocalStorage
- useMousePosition
